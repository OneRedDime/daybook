===============
 daybook-clear
===============

--------------------------
Clear a ledger in daybookd
--------------------------

.. include:: _manual-section.rst

SYNOPSIS
========

**daybook** [global-opts] **clear** [options]

DESCRIPTION
===========
This command commands the target instance of daybookd to clear the user's
ledger. The parameters related to daybookd and user credentials can be found in
~/.config/daybook/daybook.ini .

OPTIONS
=======
These options must be specified after the subcommand.

**-h**, **--help**
        Display a help message and exit.
